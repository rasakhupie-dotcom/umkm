Panduan Setup Apps Script

1
Upload & Buka Google Sheets
Upload Warkop_Stok_Opname_System.xlsx ke Google Drive → klik kanan → Buka dengan Google Sheets
2
Buka Apps Script Editor
Di Google Sheets → klik menu Extensions → Apps Script
3
Paste Kode Code.gs
Hapus kode default → paste seluruh isi file Code.gs yang ada di folder apps-script/ → ganti ownerEmail
4
Paste appsscript.json
Di editor → klik ikon ⚙️ Project Settings → centang "Show appsscript.json" → paste isi file appsscript.json (untuk OAuth scope Gmail)
5
Deploy sebagai Web App
Klik Deploy → New deployment
Type: Web app
Execute as: Me
Access: Anyone
→ Klik Deploy → Copy URL
6
Paste URL di sini
Tempel URL yang di-copy ke kolom "Web App URL" di sebelah kiri → klik Simpan & Test
7
Aktifkan Trigger Email
Di Apps Script editor → pilih fungsi setupTrigger → klik Run ▶ → authorize → email laporan aktif!
Catatan: Setiap edit Code.gs harus deploy ulang:
Deploy → Manage deployments → ✏️ Edit → New version → Deploy
